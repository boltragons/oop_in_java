# oop_in_java - OOP Codes in Java Language!

## Códigos primariamente em português.

Códigos escritos em Java para a disciplina de "Programação Orientada a Objetos", na Universidade Federal do Ceará.

1. Cada pasta contém códigos relacionados a um assunto da disciplina.
2. Códigos na subpasta src/.
3. Documentação na subpasta doc/. Procure por "index.html".
4. Funções principais sob o nome de "Solver.java".
